# メディア表現入門演習Ⅱ 発表順決定プログラム

メディア表現学科1年次「メディア表現発展演習Ⅱ」の発表順を決定・表示するWebアプリケーションです。

## 機能

- CSVファイルから発表グループの情報を読み込み
- 発表順をランダムにシャッフル（ブロック方式）
- 発表順の確定とJSONファイルでの保存
- 確定後のスケジュール表示
- 各グループの発表資料へのリンク

## ファイル構成

- `index.html` - メインHTMLファイル
- `style.css` - スタイルシート
- `app.js` - メインJavaScriptファイル
- `data.csv` - 発表グループのデータ（CSV形式）

## GitHub Pagesでの公開方法

このリポジトリにはGitHub Actionsのワークフローが設定されています。以下の手順でGitHub Pagesを有効化してください：

1. このリポジトリをGitHubにプッシュします
2. リポジトリの「Settings」→「Pages」に移動
3. 「Source」で「GitHub Actions」を選択
4. 「Save」をクリック
5. `main`ブランチにプッシュすると、自動的にGitHub Pagesにデプロイされます
6. 数分後に `https://[ユーザー名].github.io/[リポジトリ名]/` でアクセス可能になります

**注意**: 初回デプロイには数分かかる場合があります。デプロイの進行状況は「Actions」タブで確認できます。

## 使用方法

1. `data.csv`に発表グループの情報を記載
2. ブラウザで`index.html`を開く、またはGitHub Pagesで公開
3. 「発表順を確定」ボタンをクリックして順序を確定
4. 確定後、`order.json`がダウンロードされます
5. 次回以降は`order.json`をアップロードすることで、確定済みの順序を復元できます

## データ形式

### data.csv

```csv
class,group,theme,presenters,materials_url
高尾クラス,グループA,テーマ名,発表者1;発表者2,URL
```

- `class`: クラス名（高尾クラス、八尾クラス、山下クラス）
- `group`: グループ名
- `theme`: 発表テーマ
- `presenters`: 発表者名（セミコロン区切り）
- `materials_url`: 発表資料のURL

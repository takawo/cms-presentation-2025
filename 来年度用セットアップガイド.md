## 必要な素材一覧

### 必須素材

- ✅ **CSVデータファイル** - 各グループの情報（クラス名、グループ名、テーマ、プレゼンURL、資料URL、PDF URL）

## データファイル（CSV）の準備

### CSVファイルの形式

`data/data_5classes.csv`ファイルを作成し、以下の形式でデータを記入します。

```csv
class,group,theme,presentation_url,materials_url,pdf_url
```

### 各カラムの説明

| カラム名 | 説明 | 必須 | 例 |
|---------|------|------|-----|
| `class` | クラス名 | ✅ | `水野クラス`、`井口クラス`、`高尾クラス`、`八尾クラス`、`山下クラス` |
| `group` | グループ名 | ✅ | `グループA`、`グループB`、`グループC` または `1班`、`2班`、`3班` |
| `theme` | 発表テーマ | ✅ | `ケア労働が社会進出の役立つのか？` |
| `presentation_url` | プレゼン画像/動画URL | ✅ | 画像パス（セミコロン区切り）またはYouTube URL |
| `materials_url` | 発表資料URL | ⚠️ | Dropbox、SharePoint、NotionなどのURL |
| `pdf_url` | PDFファイルパス | ⚠️ | `pdf/[クラス名]_group_[A/B/C].pdf` |

### presentation_urlの形式

#### パターン1: 画像ファイル（複数可、セミコロン区切り）

```
images/mizuno_group_a/mizuno_group_a_1.jpeg;images/mizuno_group_a/mizuno_group_a_2.jpeg;...
```

- 複数の画像をセミコロン（`;`）で区切る
- マウス位置に応じて画像が切り替わる機能が有効になります

#### パターン2: YouTube動画URL

```
https://youtu.be/KiGqlalU6UI
```

または

```
https://www.youtube.com/watch?v=KiGqlalU6UI
```

- YouTubeのURLをそのまま記入
- 自動的に埋め込み形式に変換されます

### CSVファイルの例

```csv
class,group,theme,presentation_url,materials_url,pdf_url
水野クラス,グループA,ケア労働が社会進出の役立つのか？,images/mizuno_group_a/mizuno_group_a_1.jpeg;images/mizuno_group_a/mizuno_group_a_2.jpeg,https://www.dropbox.com/...,pdf/mizuno_group_a.pdf
高尾クラス,グループA,女性偉人かるたの提案,https://youtu.be/KiGqlalU6UI,https://konanwu.sharepoint.com/...,pdf/takawo_group_a.pdf
```

## 画像素材の準備

### プレゼンテーションスライド画像

各グループのスライドを画像として保存します。

#### フォルダ構造

```
images/
├── [クラス名]_group_a/
│   ├── [クラス名]_group_a_1.jpeg
│   ├── [クラス名]_group_a_2.jpeg
│   └── ...
├── [クラス名]_group_b/
│   └── ...
└── [クラス名]_group_c/
    └── ...
```

#### ファイル命名規則

- **フォルダ名**: `[クラス名のローマ字]_group_[A/B/C]`
  - 例: `mizuno_group_a`、`iguchi_group_b`、`takawo_group_c`
- **ファイル名**: `[フォルダ名]_[連番].[拡張子]`
  - 例: `mizuno_group_a_1.jpeg`、`mizuno_group_a_2.jpeg`

#### クラス名のローマ字表記

| クラス名 | ローマ字表記 |
|---------|------------|
| 水野クラス | `mizuno` |
| 井口クラス | `iguchi` |
| 高尾クラス | `takawo` |
| 八尾クラス | `yao` |
| 山下クラス | `yamashita` |

#### 画像形式

- **推奨形式**: JPEG（`.jpeg`または`.jpg`）
- **サイズ**: 適度なサイズ（1枚あたり500KB〜2MB程度）
- **アスペクト比**: 16:9推奨（スライドの標準比率）

#### 複数画像の使用方法

1. スライドを順番に画像として保存
2. CSVの`presentation_url`にセミコロン区切りで記入
3. マウスを画像上で左右に動かすと、スライドが切り替わります

## YouTube動画の設定

### トップページの紹介動画

`index.html`の48行目付近を編集します。

```html
<iframe src="https://www.youtube.com/embed/[動画ID]" frameborder="0"
    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
    allowfullscreen class="theme-youtube-iframe">
</iframe>
```

#### YouTube動画IDの取得方法

1. YouTube動画のURLから取得
   - `https://www.youtube.com/watch?v=mpX3mbleGeY` → 動画ID: `mpX3mbleGeY`
   - `https://youtu.be/mpX3mbleGeY` → 動画ID: `mpX3mbleGeY`
2. 埋め込みURL形式: `https://www.youtube.com/embed/[動画ID]`

### 各グループのプレゼン動画

CSVファイルの`presentation_url`にYouTube URLを記入するだけです。

```csv
高尾クラス,グループA,女性偉人かるたの提案,https://youtu.be/KiGqlalU6UI,...
```

## 背景画像の準備

### プレゼン中の記録写真

`images/presentation_images/`フォルダに、プレゼン中の記録写真を配置します。

#### ファイル命名規則

```
presentation_img_1.jpeg
presentation_img_2.jpeg
...
presentation_img_49.jpeg
```

- **ファイル名**: `presentation_img_[連番].jpeg`
- **推奨枚数**: 約50枚（30枚以上推奨）
- **形式**: JPEG（`.jpeg`）

#### 機能

- 背景にランダムで表示されます
- 5秒ごとに自動的に切り替わります
- ディゾルブ効果（1秒のフェードイン/アウト）で切り替わります
- ぼかし効果（4px）が適用されます
- 透明度30%で表示されます

#### JavaScriptでの自動読み込み

`js/index.js`の`initBackgroundImages()`関数で、1から49までの画像を自動的に読み込みます。

```javascript
for (let i = 1; i <= 49; i++) {
    imagePaths.push(`images/presentation_images/presentation_img_${i}.jpeg`);
}
```

**注意**: 画像の枚数が異なる場合は、この数値を変更してください。

```javascript

```

- 切り替え間隔を変更する場合は、`5000`（ミリ秒）を変更

### CSSの設定（style/style.css）

#### 背景画像のぼかし（57行目）

```css

```

- ぼかしの強さを変更する場合は、`4px`を変更

  - 薄く: `0.2`、`0.25`
  - 濃く: `0.4`、`0.5`


## トラブルシューティング

### 画像が表示されない

- ファイルパスが正しいか確認
- ファイル名の大文字小文字が一致しているか確認
- 画像ファイルが実際に存在するか確認

### YouTube動画が表示されない

- 動画IDが正しいか確認
- 動画が公開設定になっているか確認
- 埋め込みが許可されているか確認

### CSVが読み込まれない

- CSVファイルの文字コードがUTF-8か確認
- CSVファイルのパスが正しいか確認
- カラム名が正しいか確認（`class,group,theme,presentation_url,materials_url,pdf_url`）

### 背景画像が切り替わらない

- `images/presentation_images/`フォルダに画像が存在するか確認
- ファイル名が`presentation_img_[連番].jpeg`の形式か確認
- JavaScriptのコンソールでエラーがないか確認
